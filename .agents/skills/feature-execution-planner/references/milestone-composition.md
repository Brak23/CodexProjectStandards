# Milestone composition

A milestone is a validated release composition over approved criteria. It owns no product acceptance criteria and may not introduce acceptance prose.

A behavioral milestone borrows active AC revisions. An enabling milestone supports downstream releases without claiming AC completion. A maintenance milestone claims approved maintenance objectives.

The milestone owns composition-level evidence, observability, rollout, rollback or forward recovery, produced guarantees, consumed guarantees, promotion prerequisites, and release ownership.

Every current execution belongs to exactly one current milestone. Every active AC is claimed by exactly one current behavioral milestone.
