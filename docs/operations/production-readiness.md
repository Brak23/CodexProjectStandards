# Production readiness checklist

## Ownership

- [ ] Product owner
- [ ] Technical owner
- [ ] On-call and escalation
- [ ] Dependency owners
- [ ] Security contact

## Reliability

- [ ] SLIs and SLOs
- [ ] Capacity and load validation
- [ ] Timeouts, retries, idempotency, and backpressure
- [ ] Graceful degradation
- [ ] Dependency-failure testing
- [ ] Mixed-version behavior

## Data

- [ ] Backup and restore tested
- [ ] RPO and RTO documented
- [ ] Migration and reconciliation validated
- [ ] Retention and deletion implemented
- [ ] Audit events defined

## Operations

- [ ] Deployment, rollback, incident, and troubleshooting runbooks
- [ ] Dashboards, alerts, logs, and traces
- [ ] Feature flag or kill switch
- [ ] Post-deployment validation
- [ ] Disaster recovery procedure
