# Feature Execution Planner package

This package implements the repository's Gate 1 decision model, Gate 2 plan model, deterministic planning validation, generated review views, impact analysis, and implementation-authorization contracts.

Normative records are strict JSON. `decisions.md`, `plan.md`, and graph files are generated views. Approval is derived from authenticated repository review over an exact record hash; source records never claim that they approved themselves.

Use the stable Task commands documented in the root `Taskfile.yml`.
