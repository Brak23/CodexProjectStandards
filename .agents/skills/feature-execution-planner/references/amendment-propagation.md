# Amendment propagation

An approved record amendment propagates through explicit dependency graphs:

intent revision → decision closure → obligation and execution closure → milestone composition closure → authorization and promotion impact.

Changed decision revisions and their transitive dependents require revalidation. A replacement execution never inherits prior evidence automatically. A changed milestone composition creates a new milestone revision.

Intent amendments may add, revise, retire, split, or merge AC identities. The planner detects the impact but cannot author the intent change. New AC placement into a milestone requires human Gate 2 approval and is never automatic.

Uncertain isolation widens the pause. It never silently preserves authorization.
